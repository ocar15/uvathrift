Team project from the Fall 2025 semester.

I contributed most significantly as the DevOps lead for this project, which entailed:

- Configuring and initializing the Django app.
- Setting up Heroku hosting and connecting our app.
- Implementing AWS S3 bucket storage including user image uploads.

Separately, I took on building the "profile" corner of the app. This meant building the profile dashboard, designing profile editting forms, and implementing the user profile photo upload and thumbnail system.

All of us on the team gathered requirements from local UVA students and met biweekly for sprint reports where we reviewed our individual progress and assigned tickets for the following sprint cycle.
